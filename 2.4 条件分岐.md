# 2.4 条件分岐
前回までで、変数の利用方法がわかりましたので、プログラムの制御について扱って行きます。プログラムの制御は、大きく分けて`条件分岐`と`繰り返し処理`です。

まずは、`条件分岐`をみていきましょう。

## 2.4.1 IF(もしXXXなら、●●●する)
まずは、条件分岐の基本をみていきます。`もしXXXなら、●●●する`とうい処理です。

例えば、もし、テストの点数が100点なら、「合格です！！」と表示すると言う処理です。実際にコードをみてみましょう。

### コード
```java
int score = 100;

if (score == 100){
  System.out.println("合格です");
}
```

### 実行結果
```
合格です
```

### 補足説明
このプログラムは、`if (score == 100)`の部分が、`変数scoreの部分が100だったら`と言う条件になります。（＝を二つかく理由は、次のせつで説明します）

この条件に当てはまる時に、その後の`{ }`の中が実行されます。今回なら、`System.out.println("合格です");`の部分が実行されます。

今回、変数scoreには100が入っているので、条件に当てはまり、「合格です」と表示されます。

仮に、変数scoreが100ではない場合、例えば99である場合は、条件に当てはまらないので、`{ }`内の処理は行われず、何も表示されません。

## 2.4.2 条件として記載できるもの
では、ifの条件として何が記載できるのでしょうか？具体的に説明して行きます。

### 一致する場合（==）
右と左の内容が一致する場合に、`true`となります。

例えば、以下のようなものです。

```java
if(a == 100){

}
```

この場合、aが100の場合に`true`と判定されます。

なお、`=`ではなく`==`となることに注意必要です。`=`としてしまうと、左右が一致するかではなく、左に右の値を格納し、それが成功したら`true`、失敗したら`false`となります。つまり、通常値の格納に失敗すると言うことはないため、必ず`true`になってしまいます。

### 〜より小さい、〜より大きい
`<`や、`>`を使用してあら渡します。

```java
if (a > 100) {
}
```

こうすると、a が 100 より大きければ`true`となります。

```java
if (a < 100) {
}
```

この場合は、a が 100より小さい場合に`true`となります。

なお、以上、以下の関係性を示したい場合、`<=`、`>=`と記載することで実現できます。

```java
if (a => 100) {
}
```
この場合、a が 100以上の場合に`true`となります。

```java
if (a >= 100) {
}
```
この場合は、a が 100以下の場合に`true`となります。

### 否定
〜の条件に一致しない、と言う条件も記載できます。

例えば、aが100ではない場合のみ、何か処理を行うと言うことです。
その場合、`!=`と記載します。

```java
if (a != 100) {
}
```

### AND条件
二つの条件を結合して、条件判定することもできます。結合するときは、どちらも満たす場合（`AND`条件）、どちらか片方のみを満たす場合（`OR`条件）のパターンがあります

AND条件の場合、`&&`で条件を結合します。

例えば、変数aとbがあったとして、aが100、bが0の時のみ何か処理をしたいと言うような時、以下のようにコードを書きます。

```java
if (a == 100 && b == 0){
}
```

### OR条件
OR条件として複数条件を扱いたい場合、`||`として記号を記載します。

例えば、変数aとbがあり、aとbのどちらか片方が100出会った時のみ何か処理をしたい、と言う場合は以下のように記載します。

## 2.4.3 else
例えば、ifの条件を満たさない場合は、別の処理を行う、と言うことを記載できます。

例えば、テストの点数が100点であれば「合格」、それ以外の場合は、「もっと頑張りましょう」と表示させるような処理です。

具体的なコードをみてみましょう。

```java
int score = 100;

if (score == 100){
  System.out.println("合格です");
} else {
  System.out.println("もっと頑張りましょう");
}
```

このように記載した場合、scoreが100であれば一つ目の`{ }`の中の`System.out.println("合格です");`が実行され、それ以外の時は二つ目の`{ }`の中の`System.out.println("もっと頑張りましょう");`が実行される形になります。

## 2.4.4 else if
例えば、ifの条件を満たさない場合、さらに条件を指定して判定を行うことができます。`else if`と言う記載を行います。

例えば、`score`の値が100であれば「合格です」、100ではなく、80より大きい場合は、「後一息です」と表示するような処理です。

コードとしてはこのように記載します。
```java
int score = 100;

if (score == 100){
  System.out.println("合格です");
} else if(score >= 80) {
  System.out.println("あと一息です。");
}
```

このように記載した場合、scoreが100であれば一つ目の`{ }`の中の`System.out.println("合格です");`が実行され、それ以外の時、80以上であれは二つ目の`{ }`の中の`System.out.println("あと一息です。");`が実行される形になります。79以下の場合は、何も実行されません。

79以下の場合に「もっと頑張りましょう」と表示させたければ、先ほどのelseを組み合わせることができます。

```java
int score = 100;

if (score == 100){
  System.out.println("合格です");
} else if(score >= 80) {
  System.out.println("あと一息です。");
}else {
  System.out.println("もっと頑張りましょう");
}
```

else ifの条件は、複数記載できます。例えば、79以下の場合でも、40以上であれば、「もっと頑張りましょう」、39以下であれば「赤点です」と言うような処理も以下のように記載できます。

```java
int score = 100;

if (score == 100){
  System.out.println("合格です。");
} else if(score >= 80) {
  System.out.println("あと一息です。");
}else if(score >= 40) {
  System.out.println("もっと頑張りましょう。");
} else{
  System.out.println("赤点です。");
}
```

## 2.4.5 チュートリアル
では、実際に計算処理のプログラムを作成し、動かしてみましょう。

まずは、新しいクラスを作成しましょう。名前は`IfSample.java`としておきましょう。

ここに、ソースコードを書いて行きます
今回は、`各変数を宣言し、値を格納して、それを表示するコード`を書いてみます。

記載するコードは以下のようになります。

### IfSample.java
```java
public class IfSample {
  public static void main(String[] args) {
    System.out.println("if elseif elseのサンプル");
    int score = 100;

    if (score == 100) {
      System.out.println("合格です。");
    } else if (score >= 80) {
      System.out.println("あと一息です。");
    } else if (score >= 40) {
      System.out.println("もっと頑張りましょう。");
    } else {
      System.out.println("赤点です。");
    }

    System.out.println("And条件、OR条件のサンプル");
    int value1 = 100;
    int value2 = 0;

    if (value1 == 0 && value2 == 0) {
      System.out.println("value1とvalue2がどちらも0です");
    } else if (value1 == 0 || value2 == 0) {
      System.out.println("value1とvalue2のどちらかが0です");
    }

    System.out.println("否定条件サンプル");
    if (score != 0) {
      System.out.println("socreは100ではありません");
    }
  }
}
```

ソースコードは、なるべく書き写してみましょう。
これを写経と呼ぶことがありますが、プログラム学習中は、コピペするより、写経した方が理解が深まります。

コードが記載できたら、実行してみましょう。
実行結果が以下のようになれば、成功です。

### 実行結果

```
if elseif elseのサンプル
合格です。
And条件、OR条件のサンプル
value1とvalue2のどちらかが0です
否定条件サンプル
socreは100ではありません
```
